This map demonstrates data about litterbins located near Melbourne Australia

Each color represents different kinds of litterbins that were classified in the GeoJson file.

  • Red represents OS-Local litterbins.
  • Green represents Primary Access Road litterbins.
  • yellow represents OS-Regional litterbins.
  • Blue represents OS-State litterbins.
  • orange represents walkway litterbins.
  • purple represents OS-Neighbourhood litterbins.
  • pink represents no code allocated litterbins.
